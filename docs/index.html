Google Antigravity Setup

The quick install guide for students.

Important Prerequisites

  • Environment: First, spin up a Linux CloudTop via internal corp tools and wait until you reach the graphical desktop.
  • Account: You MUST use a consumer Gmail account (your corporate account will not work). Ensure you have the password and your 2FA device handy for this account.
1

Open Your Terminal

Once you are on your CloudTop desktop, open the terminal app. You can do this quickly by pressing Ctrl + Alt + T.

2

Copy the Install Command

Click the button below to copy the automated installation script. This single command downloads the installer and runs it entirely in memory—no extra execution permissions required!

curl -sL "https://raw.githubusercontent.com/wtg-codes/agv-easy-install/main/antigravity-manager.sh" | bash
3

Run and Select Your Installation

Go back to your terminal, paste the command you just copied (Ctrl + Shift + V), and press Enter.

Menu Options Explained:

💡 Always try Option 1 first! If that errors out, just run the command again and choose Option 2 instead.

  • 1
    Standard Repository (Recommended)

    Installs the app securely via your system's package manager so it stays updated automatically.

  • 2
    Standalone Tarball (Fallback)

    If Option 1 fails, use this. It downloads the IDE to a local folder (no sudo required).

  • 3
    Install/Update this Manager script

    Saves this menu to your computer so you can access it anytime by typing antigravity-manager.

  • 4
    Remove/Uninstall setup

    Safely deletes the IDE and shortcuts (keeps your actual code safe).

  • 5
    Remove Manager script

    Deletes the antigravity-manager tool from your system.

Launch Antigravity

You're all set! You can now launch Google Antigravity directly from your Ubuntu Application Menu, or double-click the shortcut it placed on your Desktop.

Fallback: Running from the Terminal

If the icons don't appear right away, you can always launch it from your terminal by typing:

antigravity

If it says "command not found", close and reopen your terminal to refresh your system path, then try again.

Where are my files? • Workspace: ~/my-antigravity-work
• If you chose Option 1 (Standard): Installed system-wide via APT/DNF.
• If you chose Option 2 (Tarball): Installed to ~/.local/lib/antigravity.

Managing Your Installation

The script automatically saved a management tool to your system! If you ever need to uninstall the IDE, switch installation types, or force an update, just open a terminal and type: antigravity-manager
