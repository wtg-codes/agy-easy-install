Google Antigravity Setup

The 60-second quick install guide for students.

1

Open Your Terminal

On your Ubuntu machine, open the terminal app. You can do this quickly by pressing Ctrl + Alt + T.

2

Copy the Install Command

Click the button below to copy the automated installation script. This single command downloads the installer and runs it entirely in memory—no extra execution permissions required!

curl -sL "https://raw.githubusercontent.com/wtg-codes/agv-easy-install/main/antigravity-manager.sh" | bash
3

Paste and Run

Go back to your terminal, paste the command you just copied (Ctrl + Shift + V), and press Enter. The script will download the IDE, set up your ~/my-antigravity-work workspace, and register the app automatically.

Launch Antigravity

You're all set! You can now launch Google Antigravity directly from your Ubuntu Application Menu, or double-click the shortcut it placed on your Desktop.

Fallback: Running from the Terminal

If the icons don't appear right away, you can always launch it from your terminal by typing:

antigravity

If it says "command not found", close and reopen your terminal to refresh your system path, then try again.

Direct Execution Fallback

If the above still fails, navigate directly to the folder and run it manually:

cd ~/.local/bin
./antigravity

Where are my files? • App installation: ~/.local/lib/antigravity
• Your workspace: ~/my-antigravity-work
